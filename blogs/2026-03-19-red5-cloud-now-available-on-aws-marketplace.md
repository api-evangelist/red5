<p><a href="https://aws.amazon.com/marketplace/pp/prodview-3aihdcp4nexuw" rel="noopener" target="_blank">Red5 Cloud is now available on AWS Marketplace</a>, making it easier for AWS customers to deploy ultra-low-latency live streaming without managing infrastructure.</p>



<p>Unlike traditional Marketplace listings that require customers to install and run software inside their own AWS accounts, <a href="https://www.red5.net/red5-cloud-low-latency-live-streaming-platform/">Red5 Cloud</a> remains fully managed. You subscribe through AWS Marketplace, create your Red5 Cloud account, and start streaming. We handle the infrastructure, scaling, monitoring, and performance optimization behind the scenes.</p>

<h2 class="wp-block-heading" id="what-this-means-for-aws-customers">What This Means for AWS Customers</h2>



<p>AWS Marketplace provides a streamlined path for procurement, billing consolidation, and contract alignment. By listing Red5 Cloud on AWS Marketplace, we make it possible to:</p>



<ul class="wp-block-list">
<li>Centralize billing under your AWS account.</li>



<li>Apply AWS credits or committed spend.</li>



<li>Faster procurement.</li>



<li>Maintain compliance within your existing cloud governance model.</li>
</ul>



<p>If your organization has an existing AWS Enterprise Agreement or committed cloud spend, you can use your AWS credits toward Red5 Cloud usage. That means you can launch and scale live streaming workloads without introducing a new vendor approval cycle or separate billing process.</p>



<h2 class="wp-block-heading" id="red5-cloud-on-aws-marketplace">Red5 Cloud on AWS Marketplace</h2>



<p>Red5 Cloud delivers a fully managed, globally distributed streaming platform built for <a href="https://www.red5.net/blog/real-time-data-streaming-for-live-video/">interactive</a>, real-time applications.</p>



<h3 class="wp-block-heading" id="features">Features</h3>



<ul class="wp-block-list">
<li><strong>Real-Time Streaming:</strong> Deliver video, audio, and data with near-zero latency, crucial for interactive and live content, including gaming, <a href="https://www.red5.net/solutions/sports-live-streaming/">live sports</a> events, and real-time communication.</li>



<li><strong>Scalable Infrastructure:</strong> Red5 Cloud can scale easily to meet the needs of growing audiences, providing dynamic provisioning and resource management to ensure consistent performance during high-demand periods.</li>



<li><strong>Support for Multiple Streaming Protocols:</strong> Red5 Cloud supports a wide array of <a href="https://www.red5.net/video-streaming-technology/">protocols</a>, including <a href="https://www.red5.net/webrtc-server/">WebRTC</a>, <a href="https://www.red5.net/rtmp-server/">RTMP</a>, <a href="https://www.red5.net/srt-streaming/">SRT</a>, <a href="https://www.red5.net/zixi-protocol/">Zixi</a>, <a href="https://www.red5.net/rtsp-protocol/">RTSP</a>, <a href="https://www.red5.net/hls-server/">HLS</a>, <a href="https://www.red5.net/whip-and-whep/">WHIP and WHEP</a>, and more, ensuring compatibility with virtually any device or platform.</li>



<li><strong>Robust Recording Capabilities: </strong>Record and store your live streams effortlessly for on-demand playback.</li>



<li><strong>Enhanced Security: </strong>Protect your streams with advanced security features such as encryption, access control, and token-based authentication to safeguard your content from unauthorized access.</li>



<li><strong>Real-Time Analytics: </strong>Utilize built-in analytics and reporting tools to track stream performance, monitor viewer engagement, and gain insights into bandwidth usage and stream health.</li>



<li><strong>Advanced Transcoding Options: </strong>Convert streams into the right formats and bitrates in real-time. Red5 Cloud offers adaptive bitrate streaming and <a href="https://www.red5.net/blog/what-is-a-codec/">codec</a> conversion to ensure smooth, high-quality playback on any device under any network conditions.</li>



<li><strong>Frame-Accurate Metadata Syncing: </strong>Send real-time metadata with your streams, including speaker names, song titles, scores, and captions, to power overlays, discovery, and interactive features.</li>



<li><strong>Extensive API Integration With Webhooks And Full-Featured SDKs: </strong>Automate workflows with flexible APIs and event-driven webhooks, and build custom apps using <a href="https://www.red5.net/live-streaming-sdks/">SDKs</a> for iOS, Android, HTML, Unreal Engine, Windows, macOS, Linux, and Unity.</li>



<li><strong>Compositing: </strong>Mix and arrange multiple video and audio feeds in real-time using server-integrated mixers, choosing <a href="https://www.red5.net/docs/red5-pro/users-guide/mixer/brew-mixer-overview/">Brew</a> for native performance or CEF for web-based flexibility.</li>
</ul>



<h3 class="wp-block-heading" id="benefits">Benefits</h3>



<ul class="wp-block-list">
<li><strong>Operational simplicity:</strong> You do not deploy or maintain streaming clusters in your AWS account. Red5 Cloud handles all infrastructure, scaling, and global delivery so you can focus on building products and creating content.</li>



<li><strong>Scalable Infrastructure:</strong> Red5 Cloud can scale easily to meet the needs of growing audiences, providing dynamic provisioning and resource management to ensure consistent performance during high-demand periods.</li>



<li><strong>Faster procurement:</strong> Subscribe directly through AWS Marketplace and align usage with your AWS billing.</li>



<li><strong>Use existing AWS credits:</strong> If you have a large AWS commit, you can apply your AWS credits toward Red5 Cloud streaming usage.</li>



<li><strong>Cost-Efficient:</strong> Red5 Cloud offers flexible plans for developers, startups, and enterprises, from small projects to large deployments serving millions. Deploy infrastructure only when you need it, reduce always-on costs, and scale affordably without sacrificing performance or security.</li>



<li><strong>Reduced internal friction:</strong> Legal, procurement, and finance teams already familiar with AWS Marketplace can approve faster.</li>
</ul>



<h3 class="wp-block-heading" id="how-to-get-started">How to Get Started</h3>



<ol class="wp-block-list">
<li>Visit the <a href="https://aws.amazon.com/marketplace/pp/prodview-3aihdcp4nexuw" rel="noopener" target="_blank">Red5 Cloud listing on AWS Marketplace</a>.</li>



<li>Click View purchase options, choose a plan in Dimensions, and click Subscribe.</li>



<li>Go to your subscription and click Set up account to create your Red5 Cloud account.</li>



<li>Launch your streaming instance and begin broadcasting.</li>
</ol>



<p>There is no software to install and no infrastructure to manage. Once subscribed, you can start streaming immediately.</p>
